# futebol-forum-app

Author: Ajax Tchakounte

## Project overview

This project is a solution using the 3-Tier architecture that lets users register and log in to a forum, where users may ask and answer questions. It includes:

1. A database schema and some example data. For demoing purposes, choose a free online database service, or create example files that load an empty database with example data.

2. An application layer written using node.js and express.js.

3. A single page application (client-side rich web application) that communicates with the application layer by sending and receiving JSON data.

GitHub link: [https://github.com/atchakounte/futebol-forum-app](https://github.com/atchakounte/futebol-forum-app)
GitHub clone link: [https://github.com/atchakounte/futebol-forum-app]

## User stories

AS A user, I want to be able to:

- Search for topics related to futebol
- View forum topics
- Post questions to forum topics
- Post answers to forum topics
- view topic questions
- view topic answers

## Technical functionality

From the technical side:

- The main page contains Topic Lists links to view each Topic
- The main page contains links to forms page for submitting Topic quesions and answers
- user input in form subission
- parse the response
- display data
